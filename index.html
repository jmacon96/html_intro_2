HTML

  1. HTML is the foundation of web development. This is the language that provides the structure and content of a website. A website can be created with just html, but it won’t be the most interesting website in the world as you will soon see.
  2. HTML has 2 types of tags (commands): self closing and non self closing
    • A self closing tag is a tag that is enclosed in one set of brackets i.e., the tag does not have a corresponding closing tag. An example of a self closing tag would be <hr>, which adds a horizontal reference line.
    • A non self closing tag is a tag that needs a corresponding closing tag. The effects of the tag will be applied to everything between the tag and closing tag. A closing tag is simply the tag with “/” added after the opening angle bracket. An example of a non self closing tag is <h1>Header Text </h1> which creates a header with the text “Header Text”. If the tag isn’t closed, everything after the initial tag will have its effect applied. For example: <h1>Header Text <p>Paragraph will create a header with two lines. The first line is “Header Text” and the second line will be “Paragraph”, and both lines will have the h1 effects applied. This is not an error, and can be done intentionally to create headers with multiple lines. However, there is an obvious difference between the above example and this example: <h1>Header Text</h1> <p>Paragraph</p>. This example creates a single line header with the text “Header Text”, followed by a paragraph on the next line.
  3. Look over HTML Basic Tags file. This document has everything you need to create your first website with usage examples. Remember that you can have tags inside of other tags.
