Applying CSS to Your Website

  1. This will be done by utilizing external CSS. In other words, the CSS code will go in a separate file within the website folder. You will have to link the CSS code to every file in your website. This will apply the CSS code to every instance of each element specified in the file
  2. In your website folder, create a new folder named css . Within the css folder, create a new file named styles.css.
  3. To link styles.css to your html code, you would need to add this line to the head section of each html file after the title: <link rel="stylesheet" href="css/styles.css">. (The href address will need to be altered if you named the folder or file a different name)
  4. In styles.css add some color to some of the elements in your website. Feel free to free style it or if you want to look at color palettes go here
  5. Modify the <hr> elements as well to add some extra styling go here
  6. My code is included, so if you get stuck, you can look at it. My files that have the CSS applied end in CSS.
