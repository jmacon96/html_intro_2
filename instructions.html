Creating Your Website

  1. Create a new folder in your web development folder for this website.
  2. Inside atom, navigate to your website folder and create a new file named index.html. This will be your home page.
  3. If you installed all of the packages in atom listed above you can just type html and press enter in atom to create the html boilerplate code. If you didn’t or that didn’t work, I will provide the boilerplate code as well. The boilerplate code is a skeleton that contains everything you need to create a html page (minus the code to add content of course).
    • In the boilerplate code, there will be a <head> tag with a <title> tag inside. Inside of the <title> will be whatever you want to name your website (maybe something like My Website). This title will appear on the tab in your browser when you open your website.
    • There will also be a <body> tag in the boilerplate. This is where the code for the content of your website will go.
