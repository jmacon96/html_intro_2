Hints and Stuff

  1. Hint: The section with the picture and name / other information is inside of a table
  2. Hint: The stars in the skills section are unicode emojis. They (and other emojis can be found here
  3. Hint: The My Hobbies and Contact Me!!! pages are links to other pages. You can create separate html files similar to index.html within your website folder. If you do this, you can replace the link to a website with the new file name. Refer to the HTML Basic Tags document for the tag to create a link
  4. Hint: You do not need to recreate the Fun Stuff page. That was done for experimental purposes.
  5. Hint: The Skills Section is a little complicated, so it will be considered extra credit. Don’t worry too much if it doesn’t look right or if you only have half of the table (that was a minor hint) or if you don’t do it
  6. Hint: Everything under the horizontal rule on Contact Me!!! is part of a form. The smaller boxes are <input> with a type of text. The larger box is a <textarea> with the row and col size specified. Everything is in HTML Basic Tags. Don’t forget to include the <form> tag.
  7. I will provide the code to my website for reference if you get stuck. It would be best for you to attempt to build it without looking at the code first.
  8. Be creative!!! Make your website to your liking
  9. To view your website, open the html file in your file explorer. It will open in a web browser. Note that this is only a local website and you no one else will be able to visit it.
